## User Story

As a frontend developer,
I want a reusable business card component,
So that I can display business information consistently.

## Acceptance Criteria

- [ ] BusinessCard component created accepting business data as props
- [ ] Card displays: business name, category, description
- [ ] Contact info displayed: email (mailto link), phone (tel link), address
- [ ] Website URL displayed as clickable link (opens in new tab)
- [ ] Score badge displayed with color coding: red if score < 70, green if >= 70
- [ ] Problem indicators shown as icons (performance, SEO, accessibility) with tooltips
- [ ] Card has hover state with subtle elevation animation
- [ ] Click on card navigates to business detail view
- [ ] Component responsive and works on mobile devices
