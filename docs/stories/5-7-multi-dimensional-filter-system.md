# Story 5.7: Multi-Dimensional Filter System

**Epic:** Epic 5 - Frontend Business Cards & Discovery UI
**Status:** ‚è≥ Pending

## User Story

As a user,
I want to filter businesses by multiple criteria,
So that I can focus on specific segments of leads.

## Acceptance Criteria

- [ ] Filter sidebar created with collapsible sections
- [ ] Score range filter: slider or buttons for <70, 70-85, >85
- [ ] Location filter: dropdown or autocomplete for cities/regions
- [ ] Category filter: checkboxes for business types
- [ ] Multiple filters combine with AND logic
- [ ] Active filters displayed as removable chips
- [ ] "Clear all filters" button resets to default state
- [ ] Filter state persists in URL query parameters
- [ ] Grid updates immediately when filters change
