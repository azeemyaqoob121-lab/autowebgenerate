# Story 5.8: Business Detail View

**Epic:** Epic 5 - Frontend Business Cards & Discovery UI
**Status:** ‚è≥ Pending

## User Story

As a user,
I want to view comprehensive details for a specific business,
So that I can make informed outreach decisions.

## Acceptance Criteria

- [ ] Detail page created at /businesses/[id]
- [ ] Page displays all business information: name, category, description, contact details
- [ ] Website URL shown with "Visit Website" button (opens in new tab)
- [ ] Score breakdown visualization (circular progress bars for each category)
- [ ] Detailed problem list displayed with categorization and severity
- [ ] Evaluation timestamp shown
- [ ] "Preview Templates" button shown if templates available
- [ ] "Re-evaluate Website" button triggers new evaluation
- [ ] Loading state while data fetches
- [ ] Error handling for invalid business ID (404 page)
